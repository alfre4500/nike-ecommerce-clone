# 👟 Nike Ecommerce Clone — Premium Experience

![Nike Banner](https://images.unsplash.com/photo-1542291026-7eec264c27ff?auto=format&fit=crop&w=1200&q=80)

Una experiencia de comercio electrónico de alto rendimiento inspirada en la estética minimalista y soberbia de Nike. Este proyecto no es solo una tienda visual; es una aplicación funcional con gestión de estado avanzada, animaciones fluidas y una pasarela de pagos integrada.

🚀 **[Ver Demo en Vivo](#)** *(https://nike-ecommerce-clone-khaki.vercel.app/)*

---

## ✨ Características Principales

* **Experiencia Visual Immersiva (GSAP):** Animaciones dinámicas de entrada (stagger), efectos de flotación continua para productos y transiciones de color de fondo basadas en la imagen del calzado.
* **Gestión de Estado Robusta (Zustand):** Control total del carrito de compras y notificaciones (Toasts) sin re-renders innecesarios.
* **Persistencia de Datos:** El carrito se mantiene intacto incluso después de recargar la página gracias al middleware de persistencia en LocalStorage.
* **Pasarela de Pagos (Stripe):** Integración real con Stripe Elements para la validación y procesamiento seguro de tarjetas de crédito en modo de prueba.
* **Diseño Mobile-First:** Interfaz totalmente responsiva construida con Tailwind CSS, optimizada para la navegación táctil.
* **Feedback Visual:** Sistema de notificaciones personalizadas y efectos de confeti al completar una orden con éxito.
