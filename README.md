# Nike E-Commerce Clone 👟

Una réplica moderna, interactiva y minimalista de la tienda de Nike, desarrollada con enfoque en una experiencia de usuario (UX) premium y animaciones fluidas. 

Este proyecto está diseñado para funcionar como una aplicación nativa o "ventana" dentro de un entorno de portfolio interactivo que simula un sistema operativo de escritorio.

## ✨ Características Principales

* **Animaciones Premium (GSAP):** Transiciones fluidas, aparición en cascada de elementos (ScrollTrigger) y modales animados.
* **Gestión de Estado Global (Zustand):** Carrito de compras funcional y sincronizado en todas las vistas de la aplicación, sin prop-drilling.
* **Buscador en Tiempo Real:** Modal a pantalla completa con fondo desenfocado y filtrado instantáneo de productos.
* **Notificaciones Toast:** Sistema de alertas personalizado e integrado con animaciones para confirmar acciones del usuario.
* **Diseño "CSS-First":** Maquetación responsiva y moderna utilizando el nuevo motor de Tailwind CSS v4.
* **Enrutamiento Dinámico:** Vistas independientes para el catálogo y el detalle de cada producto usando React Router.

## 🛠️ Tecnologías Utilizadas

* **React (Vite)**
* **Tailwind CSS v4**
* **GSAP (GreenSock)**
* **Zustand**
* **React Router DOM**
* **Lucide Icons**
